# 🎨 Цветашки (Colorlets)

Простая и безопасная веб-игра для малышей (2+ лет) на изучение основных цветов. Работает прямо в браузере, не требует установки и интернета после загрузки.

🌐 **Играть онлайн:** https://krivich.github.io/colorlets/

## ✨ Особенности
- 🔊 **Голосовые задания:** игра просит найти цвет и повторяет подсказку, если ребёнок отвлёкся
- 💡 **Мягкая коррекция:** при неправильном нажатии игра называет цвет и даёт детскую ассоциацию (`"Это оранжевый, как апельсин"`)
- 📊 **Прогресс:** отслеживает, сколько цветов уже найдено, и поздравляет при завершении
- 📱 **Адаптивность:** крупные карточки, удобные для детских пальчиков, работает на телефонах, планшетах и ПК
- ⚡ **Один файл:** никаких фреймворков, сборщиков и зависимостей

## ⚙️ Настройка цветов
Все цвета, оттенки и подсказки вынесены в начало файла `index.html` в массив `COLORS_CONFIG`:
```js
const COLORS_CONFIG = [
    { name: "Красный", hex: "#EF5350", hints: ["как помидорка", "как ягодка"] },
    { name: "Розовый", hex: "#F06292", hints: ["как свинка", "как цветочек"] },
    // Просто добавьте новую строку, и игра автоматически подстроится
];
