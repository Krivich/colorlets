# 🎨 Цветашки (Colorlets)

Простая и безопасная веб-игра для малышей (2+ лет) на изучение основных цветов. Работает прямо в браузере, не требует установки и интернета после загрузки.

## 🌐 Как открыть игру
💻 **На компьютере или телефоне:** [Открыть в браузере](https://krivich.github.io/colorlets/)

## ✨ Особенности
- 🔊 **Голосовые задания:** игра просит найти цвет и повторяет подсказку каждые 6 секунд, если ребёнок отвлёкся
- 💡 **Мягкая коррекция:** при неправильном нажатии игра называет выбранный цвет и даёт ассоциацию, а затем напоминает нужный (`"Это зелёный, как лягушка. А нам нужен красный, как помидор!"`)
- 📊 **Визуальный прогресс:** кружочки внизу экрана показывают, какие цвета уже найдены
- 📱 **Адаптивность:** крупные карточки, удобные для детских пальчиков, работает на телефонах, планшетах и ПК
- ⚡ **Один файл:** никаких фреймворков, сборщиков и зависимостей

## ⚙️ Настройка цветов
Все цвета, оттенки и подсказки вынесены в начало файла `index.html` в массив `CONFIG`:
```js
const CONFIG = [
    { name: "Красный",   hex: "#E53935", hints: ["как помидор", "как клубничка", "как божья коровка"] },
    { name: "Розовый",   hex: "#EC407A", hints: ["как свинка", "как цветочек", "как фламинго"] },
    // Просто добавьте новую строку, и игра автоматически подстроится под общее количество цветов
];
